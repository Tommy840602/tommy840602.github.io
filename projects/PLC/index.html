可程式邏輯控制器研究平台（PLC）

以 PLC 通訊、控制邏輯驗證與設備資料採集為核心的研究平台，整合工業協定、後端服務與 Vue 3 即時監控介面。

Highlights

建立 PLC 資料採集與通訊測試流程，用於驗證設備點位、資料格式與連線穩定性。

整合 OPC UA、Modbus TCP 與 MQTT，模擬設備層到後端服務的資料交換與命令路由。

設計控制邏輯、Interlock 與狀態監控頁面，協助檢查異常條件與操作流程。

使用 Vue 3 建立即時監控介面，呈現設備狀態、I/O 點位、命令紀錄與告警資訊。

Engineering Focus

此專案聚焦在工業控制系統中 PLC 與上層系統的整合方式，包含設備點位建模、協定轉換、命令邊界、資料一致性與即時監控。

面試時可用來說明如何把現場設備訊號轉換成可觀測、可驗證且能與後端服務協作的工程平台。
