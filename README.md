# macOS Portfolio

Astro + Vue 3 interview portfolio with a macOS-inspired desktop entry screen.

## Commands

```bash
npm install
npm run dev
npm run build
npm run preview
```

## GitHub Pages

The workflow at `.github/workflows/deploy-portfolio.yml` builds this folder and deploys `portfolio/dist` through GitHub Pages Actions.

In the repository settings, set Pages source to **GitHub Actions**.

## Content

Project content lives in `src/data/projects.js`. The site is intentionally JavaScript-only and does not add TypeScript.
